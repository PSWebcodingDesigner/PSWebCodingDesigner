저도 영진님을 따라 예제를 한번 만들어보았습니다! 사실 제가 짠 코드는 아니고..
책에서 본 예제보고 복사 붙여넣기한 코드이지만, 따라쓰고 100% 이해하는것도 시간이 많이 걸리더라고요ㅠㅠ
제 나름대로 이해한 내용은 다음과 같습니다. @지용님 혹시 잘못되었거나 보충해주실 부분 있으시면 첨삭해주시면 감사하겠습니당! 🙏🏻

1. 버튼 '이전' '다음'에 gallery라는 이름의 함수를 넣는다.
2. 버튼 '이전'을 클릭하면 매개변수가 0, 즉 false가 되므로 else이후의 if문에 걸린다.
3. 버튼 '다음'을 클릭하면 매개변수가 1, 즉 true가 되므로 else이전 if문 num++에 걸린다.
4. 전체 이미지의 갯수가 7개 이므로 num==7일때에, 함수는 종료되어야한다. if(num==7) return;
5. num==1 일때에는 함수가 종료되는데, 파일이름이 1부터 시작하기 때문이다.
저는 이 예제를 통해 return을 확실히 이해하게 된 것 같습니다. 여러분도 참고해보세용!
